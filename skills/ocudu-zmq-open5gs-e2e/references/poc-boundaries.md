# PoC Boundaries

This lane is intentionally a proof of concept:

- `srsRAN_Project` provides the gNB side
- `srsUE` from `srsRAN_4G release_23_11` provides the prototype 5G UE path
- ZeroMQ replaces real RF hardware
- Open5GS is run from vendored compose assets

Do not market this lane as production-grade validation.
