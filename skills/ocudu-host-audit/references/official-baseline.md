# Official Baseline

- The current srsRAN Project docs describe a Linux build baseline built around `cmake`, `make`, `gcc`, `g++`, and `pkg-config`.
- The same docs call out Ubuntu packages such as `libfftw3-dev`, `libmbedtls-dev`, `libsctp-dev`, `libyaml-cpp-dev`, and `libgtest-dev`.
- Skillful RAN uses those items as a practical host-audit baseline for OCUDU-focused workstation prep.
- Official sources:
  - https://docs.srsran.com/projects/project/en/latest/user_manuals/source/installation.html
  - https://github.com/srsran/srsRAN_Project
